💹 Professional Finance Simulator~ モンテカルロ法による資産寿命の構造的解析 ~Professional Finance Simulator は、OpenSiv3Dを用いた高精度な金融シミュレーションツールです。数十年先の資産推移を、1,000パターンのランダムな市場シナリオ（モンテカルロ法）を用いて解析し、将来の破綻リスクを統計的に可視化します。🌟 プロジェクトの概要「老後2000万円問題」のような固定的な計算ではなく、市場の「ゆらぎ（リスク）」と「物価上昇（インフレ）」をシミュレーションに組み込んでいます。特に、暴落時に支出を自動的に抑制する**「ガードレール戦略」**の有無が、最終的な生存率（資産が底をつかない確率）にどう影響するかをリアルタイムに比較・検証できる点が最大の特徴です。✨ 主な機能1,000試行の同時シミュレーション:毎月のリターンとインフレ率を乱数生成し、30年分（360ステップ）のパスを一瞬で計算。インフレ調整済み実質価値計算:名目上の金額ではなく、物価上昇率を加味した「現在の購買力に換算した価値」を表示。ガードレール戦略の実装:前年の市場成績がマイナスの場合、取り崩し額を30%カットする動的な支出調整ロジックを搭載。対話的なパラメータ操作:初期資産、積立・取崩額、期待利回り、リスク（標準偏差）、インフレ率をスライダーで即座に変更。データエクスポート:試行ごとの最終資産、最大ドローダウン、破綻フラグをCSV形式で出力し、ExcelやPython等でのさらなる詳細分析が可能。🛠 技術スタックFramework: OpenSiv3D (C++20)Mathematical Model:幾何ブラウン運動をベースとした月次リターン算出$nominal\_value = (previous + cashflow) \times mReturn$$real\_value = nominal / priceIndex$Data Handling: JSON (設定保存) / CSV (結果出力)🚀 インストールと実行OpenSiv3D の環境が整ったVisual Studio等のプロジェクトを用意します。Main.cpp にソースを配置し、ビルドしてください。実行すると、即座に1,000本の資産曲線が描画されます。🕹 シミュレーションの見方青い細線: 個別のシミュレーション結果（50本を表示）。生存率: 30年後に資産が0円より大きかった確率。80%以上が安全圏の目安です。CSV出力: 全1,000パターンの詳細データを取得し、最悪のケース（ワーストシナリオ）を把握できます。📜 ライセンスMIT License「未来は確定していない。だからこそ、確率の構造を把握する必要がある。」
